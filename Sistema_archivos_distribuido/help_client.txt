Para ver la lista de archivos:
go run client.go --server=localhost:8080 --command=list

Para ver la información de un archivo específico:
go run client.go --server=localhost:8080 --command=info test_file.txt

Para editar un archivo:
go run client.go --server=localhost:8080 --command=edit test_file.txt

Para agregar un nuevo archivo:
go run client.go --server=localhost:8080 --command=add nuevo_archivo.txt
